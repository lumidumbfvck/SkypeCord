# SkypeCord
Another skeumorphic kinda theme for Discord to make it cooler. Reuses a lot of stuff from AeroCord [https://github.com/repojun/AeroCord/].
# Fastest way to try it out
..is to copy & paste the contents of SkypeCord.css into Vencord's QuickCSS.
